# contactus_form
This is a contact us form .  it save the user data into data base and also send user information through the email to the site owner.

Features :
- User-friendly web form for contact submissions.
- Data storage in a database.
- Email notifications to the site owner.
  
Prerequisites : 
Before you begin, ensure you have the following:
- A web server with PHP support (e.g., Apache, Nginx).
- A MySQL database for storing user data.
- An SMTP server or email service for sending emails.
